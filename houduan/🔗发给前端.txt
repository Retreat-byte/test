📡 后端API地址配置说明

===============================================

第1步：查看本机IP
   双击运行：查看本机IP.bat
   或手动运行命令：ipconfig

   找到 IPv4 地址，例如：192.168.1.100

===============================================

第2步：配置防火墙
   Windows防火墙：
   1. 控制面板 → Windows Defender 防火墙
   2. 高级设置 → 入站规则 → 新建规则
   3. 端口 → TCP → 8081 → 允许连接

===============================================

第3步：启动后端服务
   在IDEA中运行 HouduanApplication.java
   或使用命令：mvn spring-boot:run

   看到类似提示说明启动成功：
   "Started HouduanApplication in X.XXX seconds"

===============================================

第4步：发送给前端开发者

📁 需要发送的文件：
   整个 "前端代码" 文件夹

📡 需要告诉前端的信息：
   后端API地址：http://你的IP:8081/api
   例如：http://192.168.1.100:8081/api

   你可以直接复制以下内容发送：
   ╔════════════════════════════════════╗
   ║  后端API地址                       ║
   ║  http://你的IP:8081/api           ║
   ║  （替换为你查到的实际IP）          ║
   ╚════════════════════════════════════╝

===============================================

第5步：告诉前端如何配置

前端开发者收到文件后，只需要：
1. 运行 "配置后端地址.bat"
2. 选择 "2. 局域网访问"
3. 输入你的IP地址
4. 启动前端服务器
5. 访问 http://localhost:8080

或者手动修改：
打开 前端代码/config.js
修改第27行为：baseURL: 'http://你的IP:8081/api'

===============================================

✅ 测试连接是否成功

前端开发者可以在浏览器访问：
http://你的IP:8081/api/health

如果看到响应，说明连接正常！

===============================================

📊 架构说明

前端电脑 (8080) ←→ 后端电脑 (8081) ←→ 数据库 (3306)

- 前端：静态HTML/JS，运行在前端开发者的电脑
- 后端：Spring Boot，运行在你的电脑（端口8081）
- 数据库：MySQL，运行在你的电脑（端口3306）

===============================================

⚠️ 注意事项

1. 确保前后端在同一局域网（WiFi/有线网络）
2. 后端服务必须保持运行状态
3. 如果更换网络，IP地址可能会变化，需要重新配置
4. 防火墙设置很重要，否则前端无法访问

===============================================

❓ 前端遇到问题？

常见问题：
- 提示"网络连接失败" → 检查防火墙和IP地址
- CORS错误 → 后端已配置，一般不会出现
- 连接超时 → 检查是否在同一局域网

让前端开发者查看：
- 前端代码/配置说明.md
- 前端代码/🚀开始使用.txt
